# 📘 Assignment: Introdução à Programação em Python

## 🎯 Objetivo

Nesta tarefa você vai aprender os conceitos básicos de programação em Python: variáveis, entrada/saída, estruturas condicionais, loops e funções simples. O objetivo é escrever pequenos programas funcionais e bem documentados.

## 📝 Tarefas

### 🛠️ 1) Variáveis e Entrada/Saída

#### Descrição
Crie um programa que peça ao usuário seu nome e idade, e então exiba uma mensagem personalizada.

#### Requisitos
O programa deve:

- Pedir o nome do usuário
- Pedir a idade do usuário
- Exibir uma mensagem: "Olá, <nome>! Você tem <idade> anos."


### 🛠️ 2) Condicionais e Operadores

#### Descrição
Estenda o programa para verificar se o usuário é maior de idade (18 anos ou mais) e exibir uma mensagem apropriada.

#### Requisitos
O programa deve:

- Usar uma condicional para checar a idade
- Exibir "Você é maior de idade." quando idade >= 18
- Caso contrário exibir "Você é menor de idade."


### 🛠️ 3) Loops e Funções

#### Descrição
Implemente uma função que repita uma pergunta ao usuário até que ele digite "sair". A cada interação, conte quantas entradas válidas foram feitas e exiba ao final.

#### Requisitos
O programa deve:

- Definir uma função para ler entradas em loop
- Finalizar quando o usuário digitar "sair"
- Exibir o número de entradas feitas (excluindo o comando "sair")


## Dicas

- Teste seu código com diferentes entradas.
- Comente seu código e use nomes de variáveis descritivos.

## Entrega

Coloque todos os arquivos na pasta `assignments/intro-to-python`. Se desejar, envie também um pequeno README com instruções de uso.

---

Boa sorte e divirta-se programando! 🚀
